<Note type="warning">
    **Endianness Reminder:** All multi-byte fields (except `VarInt` and `VarLong`) use **big-endian**. On little-endian systems (e.g., x86), swap your bytes before sending!
</Note>

<Tabs defaultValue="primitive" orientation="vertical" className="gap-6">
  <TabsList className="space-x-2 bg-card p-1 rounded-lg" />

  {/* ---------- Primitive Integers ---------- */}
  <TabsContent value="primitive" className="p-4">
    <Steps>
      <Step>
        <StepTitle>What are Primitive Integers?</StepTitle>
        <StepContent>
          Fixed-size integers with a constant byte length; used for flags, IDs, positions, etc.
        </StepContent>
      </Step>
      <Step>
        <StepTitle>Endianness</StepTitle>
        <StepContent>
          Big-endian means highest-order byte first. On Node.js use <code className="inline">buffer.writeInt32BE</code> instead of LE.
        </StepContent>
      </Step>
    </Steps>

    <div className="overflow-x-auto my-6">
      | Name               | Bytes | Range                                  | Example Use                                   |
      | ------------------ | ----- | -------------------------------------- | ---------------------------------------------- |
      | **Boolean**        | 1     | `false` (0x00), `true` (0x01)         | `onGround` flag in movement packet            |
      | **Byte**           | 1     | –128 to 127                           | Pitch/yaw small offsets                        |
      | **Unsigned Byte**  | 1     | 0–255                                 | Legacy block IDs                              |
      | **Short**          | 2     | –32 768 to 32 767                     | Legacy Y coordinate                           |
      | **Unsigned Short** | 2     | 0–65 535                              | Port in Handshake packet                      |
      | **Int**            | 4     | –2³¹ to 2³¹–1                         | Entity & chunk IDs                            |
      | **Long**           | 8     | –2⁶³ to 2⁶³–1                        | Timestamps, UUID segments                     |
    </div>
  </TabsContent>

  {/* ---------- Floating-Point ---------- */}
  <TabsContent value="floating" className="p-4">
    <Steps>
      <Step>
        <StepTitle>Why Floats & Doubles?</StepTitle>
        <StepContent>
          They allow fractional values—vital for smooth movement, rotation, and physics.
        </StepContent>
      </Step>
    </Steps>

    <div className="overflow-x-auto my-6">
      | Type      | Bytes | Precision               | Common Use                   |
      | --------- | ----- | ----------------------- | ---------------------------- |
      | **Float** | 4     | 32-bit (~7 decimal dig.)| Player/Entity XYZ coordinates|
      | **Double**| 8     | 64-bit (~16 decimal dig.)| Projectile & fine physics   |
    </div>

    <Preview title="Float Example">

        tabs={{ js: { language: 'javascript', syntax: `const buf = Buffer.alloc(4);

        themeMode="dark"

    </Preview>
  </TabsContent>

  {/* ---------- Variable-Length ---------- */}
  <TabsContent value="varint" className="p-4">
    <p>VarInts/VarLongs use 7 bits per byte with MSB continuation. Small values = 1 byte.</p>

    <div className="overflow-x-auto my-6">
      | Type        | Bytes | Encodes                    | Notes                               |
      | ----------- | ----- | -------------------------- | ----------------------------------- |
      | **VarInt**  | 1–5   | 32-bit signed integer      | Continue until MSB=0                |
      | **VarLong** | 1–10  | 64-bit signed integer      | Same logic, up to 10 bytes          |
    </div>

    <Note type="info">
      To decode: loop read, <code className="inline">value |= (byte & 0x7F) << (7 * i)</code> until MSB=0.
    </Note>
  </TabsContent>

  {/* ---------- Strings & Text ---------- */}
  <TabsContent value="string" className="p-4">
    <p>All text fields are <strong>length-prefixed UTF-8</strong>. Length = VarInt count of UTF-16 units.</p>

    <div className="overflow-x-auto my-6">
      | Field                 | Prefix    | Encoding    | Limits                     |
      | --------------------- | --------- | ----------- | -------------------------- |
      | **String(n)**         | VarInt n  | UTF-8       | Up to n UTF-16 code units  |
      | **Identifier**        | VarInt    | UTF-8       | `[a-z0-9:_]` characters    |
      | **Text Component**    | NBT Tag   | JSON/NBT    | Rich chat, hover, click    |
    </div>

    <Note type="tip">
      JSON text must be valid: `{ "text": "..." }`, extra keys for `color`, `bold`, etc.
    </Note>
  </TabsContent>

  {/* ---------- Misc Types ---------- */}

      | **Angle**   | 1    | 1/256 of full rotation (360°/256 units) |
      | **UUID**    | 16   | 128-bit (MSB then LSB)                  |
    </div>
  </TabsContent>

      | **Prefixed Optional** | Boolean + X      | Presence flag then X             |
      | **Array of X**        | Contextual len   | Sequence of X                    |
      | **Prefixed Array**    | VarInt + data    | Length-prefixed list             |
      | **Fixed BitSet (n)**  | ⌈n/8⌉ bytes      | Bit flags per enum               |
      | **ID or X**           | VarInt + X?      | Registry reference or inline     |
